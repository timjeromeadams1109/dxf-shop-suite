API Reference

All endpoints are served via serverless functions at /api

Validation

POST/api/validate
Validate a DXF file against machine-specific rules
FormData: file (.dxf), machine? (machine key)

Cleanup

POST/api/clean
Auto-fix common DXF issues (blocks, contours, version, duplicates)
FormData: file (.dxf)
GET/api/download/{filename}
Download a cleaned DXF file

Nesting

POST/api/nest
Analyze nesting efficiency for multiple parts on a sheet
FormData: files[] (.dxf, multiple), quantity (int), sheet (e.g. 48"x96")

Part Library

GET/api/library
Search/list parts in the library
Query: name? (string), machine? (machine key)
POST/api/library/add
Add a part to the library
FormData: file (.dxf), name, machine, material, thickness, notes
DELETE/api/library/{part_id}
Remove a part from the library

Machines

GET/api/machines
Get all machine profiles and checklists
