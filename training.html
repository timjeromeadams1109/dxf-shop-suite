Section 1

What Is This App?

The DXF Shop Automation Suite helps you prepare drawing files for CNC cutting machines. Think of it as a spell-checker, but instead of checking your writing, it checks your design files to make sure the machine will cut them correctly.

What Is a DXF File?

A DXF file (Drawing Exchange Format) is like a blueprint saved on a computer. It tells a cutting machine exactly where to cut, punch, or bend a piece of metal. Most CAD programs (like AutoCAD, SolidWorks, or Fusion 360) can save files in this format.

The 4-Step Workflow

Check your file for problems using the Validate tab. This catches mistakes before they reach the machine.
Fix any issues automatically with the Clean tab. One click repairs the most common problems.
Plan how parts fit on a sheet of metal using the Nest tab. Less wasted material means lower costs.
Save parts to your library for reuse. Tag each part with its machine, material, and thickness.
First time here? Click "Try a Sample File" on the main page — it loads a pre-made bracket so you can explore without needing your own file.

Section 2

Validate — Check Your File

The Validate tab scans your drawing file and tells you if it's ready for the machine. It checks for common problems like unclosed shapes, wrong file versions, and unsupported geometry.

How to Use It

Pick your machine from the dropdown (optional). If you select a machine, extra checks specific to that machine will run. If you're not sure, leave it on "Any Machine."
Upload your file. Drag and drop a .dxf file onto the upload area, or click to browse. You can also click "Try a Sample File" to load a test bracket.
Click "Run Validation." The app sends your file to the server for analysis. This usually takes 1–3 seconds.
Read your results. You'll see a green "CAM READY" badge if everything looks good, or a red "ISSUES FOUND" badge if problems were detected.

Section 3

Clean — Auto-Fix Problems

Found problems in Validate? The Clean tab fixes the most common ones automatically. Upload your file, click one button, and download a fixed version.

How to Use It

Upload your file by dragging it onto the upload area or clicking to browse.
Click "Clean & Fix." The app processes your file and applies all available fixes.
Review the before/after comparison. You'll see what changed — open contours closed, blocks exploded, duplicates removed.
Download your cleaned file by clicking the download button. The cleaned file gets "_cleaned" added to its name.
Before and after comparison showing fixes applied to a DXF file

What Gets Fixed

ProblemWhat The Fix Does
Block referencesUngroups ("explodes") them into individual shapes the machine can read
Splines (curves)Converts them into small straight-line segments the machine can follow
Duplicate linesRemoves exact copies so the machine doesn't cut the same path twice
Wrong DXF versionSaves as AC1015 (the version most machines expect)
Note: The cleaned file is a new copy — your original file is never changed. Always keep your originals!
Section 4

Nest — Plan Your Cuts

The Nest tab helps you figure out how many sheets of metal you need and how much material you'll use. Upload your parts, tell it the sheet size and quantity, and get an efficiency estimate.

How to Use It

Pick your sheet size from the dropdown. Standard sizes include 48"×96" (4ft × 8ft), 60"×120" (5ft × 10ft), and others.
Set the quantity — how many copies of each part you need.
Upload your part files. You can upload multiple files at once — drag them all in together.
Click "Analyze Nesting." You'll see the estimated sheet utilization (higher % = less waste), how many sheets you need, and a table of each part's dimensions.
Nesting analysis showing 82% sheet utilization across 2 sheets

Reading the Results

FieldWhat It Means
Est. UtilizationWhat percentage of the sheet metal is actually used (higher = less waste)
Sheets NeededHow many full sheets of metal you'll need to cut all parts
Parts CountHow many unique parts were analyzed
What's a good utilization? Anything above 75% is solid. Below 60%, the app will suggest grouping with other parts or using a smaller sheet.
Section 5

Library — Save Your Parts

The Library tab lets you build a catalog of parts. Save each part with its machine, material, and thickness so you can find it later without digging through folders.

How to Add a Part

Enter a part name (e.g., "BRACKET-A-001"). Use a naming convention your shop will recognize.
Select the machine this part runs on. This helps filter parts later.
Fill in material and thickness (e.g., "A36 steel", "0.25 inches"). Optional but helpful for shop floor reference.
Upload the DXF file and click "Add to Library."

Searching and Filtering

Use the search bar to find parts by name, or the machine filter dropdown to show only parts for a specific machine. Click the refresh button (↻) to reload the list.

Deleting a Part

Click the ✕ button next to any part. You'll be asked to confirm before it's removed.

The Part Library with search, filter, and saved parts
Note: The library resets when the server restarts (cold start). For production use, connect to a database like Supabase for permanent storage.

Section 9

Frequently Asked Questions

What file types can I upload?
The Validate, Clean, Nest, and Library tabs accept .dxf files only. The Create tab's import area also supports .svg, .txt (processed in your browser), and .pdf, .jpg, .png, .doc (processed by the server).
What does "AC1015" mean?
AC1015 is the internal version code for AutoCAD 2000's DXF format. Most CNC controllers expect this version. Newer versions (AC1027, AC1032) may include features the machine can't understand, so this app always saves in AC1015.
Is my data saved permanently?
Machine profile customizations (controller names, custom checklists, notes) are saved in your browser's local storage — they persist between visits on the same device. The Part Library is stored on the server but resets on cold starts. Your uploaded files are processed and immediately discarded — nothing is stored permanently.
Can I use this offline?
Not fully. The Create tab and Machine Profiles work offline, but Validate, Clean, Nest, and Library features require an internet connection to reach the server.
How accurate is the nesting analysis?
The nesting estimate uses bounding-box calculations (the smallest rectangle around each part), not true geometric nesting. This means the actual utilization may be higher than reported if parts have irregular shapes that fit together like puzzle pieces. Use dedicated nesting software (like SigmaNEST or ProNest) for production-critical layouts.
What machines are supported?
12 machines across 5 types: Punch (AMADA Coma, Vella, Vipros, Apelio), Plasma (Vortman 631, Vortman 325, Lincoln), Laser (AMADA 12K Fiber), Press Brake (AMADA HR, Cincinnati, ACUSHEAR), and Shear (ACUSHEAR).
Can I add my own machine?
Not yet through the UI, but you can customize existing machine profiles by editing the controller name, adding notes, and adding custom checklist items. Custom machine creation is planned for a future update.
My file shows "ISSUES FOUND" — is it broken?
Not necessarily. Warnings (yellow) are suggestions, not hard stops. Only red errors must be fixed. Try running the file through the Clean tab — it fixes most common issues automatically. Then re-validate the cleaned file.
How do I print this guide?
Click the "Download as PDF" button at the top of this page. It opens your browser's print dialog — choose "Save as PDF" as the destination. The guide is formatted for A4/Letter paper with proper page breaks.

Glossary

TermPlain English
DXFDrawing Exchange Format — a standard file type for 2D drawings used by CNC machines
CAMComputer-Aided Manufacturing — software that turns drawings into machine instructions
CNCComputer Numerical Control — a machine controlled by a computer program
ContourAn outline or path in a drawing. "Open contour" means the path has a gap.
EntityAny shape in a drawing — a line, circle, arc, rectangle, etc.
SplineA smooth, complex curve. Most CNC machines can't follow splines — they need simple arcs and lines.
Block ReferenceA grouped set of shapes treated as one unit. Must be "exploded" (ungrouped) before the machine can read them individually.
AC1015The DXF version code for AutoCAD 2000 format — the most compatible version for CNC machines.
NestingArranging parts on a sheet of material to minimize waste — like fitting cookies on a baking sheet.
UtilizationThe percentage of the metal sheet that's actually used by parts (higher = less waste).
KerfThe width of material removed by the cutting tool (laser beam, plasma arc, etc.).
Punch PressA machine that punches holes and shapes out of sheet metal using metal tools (dies).
Plasma CutterA machine that cuts metal using a jet of superheated gas (plasma).
Laser CutterA machine that cuts metal using a focused beam of light.
Press BrakeA machine that bends sheet metal into angles and shapes.
ShearA machine that makes straight cuts across sheet metal, like a giant pair of scissors.
